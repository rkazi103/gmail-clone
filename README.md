## Run Locally

Clone the project

```bash
  git clone https://github.com/rkazi103/gmail-clone
```

Go to the project directory

```bash
  cd gmail-clone
```

Install dependencies (I used yarn)

```bash
  yarn
```

Start the server

```bash
  yarn start
```
